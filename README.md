🤖 GeniusAllWorld Bot

Многофункциональный асинхронный Telegram-бот нового поколения, построенный на базе aiogram 3.x. Проект объединяет в себе 41 уникальных модулей: от базовых утилит и нейросетей до встроенных полноценных Web Apps интерфейсов.

🛠 Технологический стек

Язык: Python 3.10+

Фреймворк: aiogram >= 3.17.0

База данных: SQLAlchemy 2.0 + aiosqlite / asyncpg

Веб-сервер: aiohttp

Фоновые задачи: apscheduler

🌌 Архитектура проекта (5 Уровней Разработки)

Проект разделен на 5 логических уровней сложности, каждый из которых добавляет новые архитектурные паттерны и возможности.



1. /start — Приветствие и описание возможностей.

2. /help — Список всех команд с описанием.

3. Эхо-бот — Бот повторяет ваше сообщение.

4. /ping — Проверка задержки отклика бота.

5. /userinfo — Вывод ID, имени и юзернейма пользователя.

6. /time — Текущее время в разных часовых поясах.

7. /chatinfo — ID чата и количество участников.

8. Смена аватара — Установка фото профиля бота через фото в сообщении.

9. /version — Вывод текущей версии кода бота.

10. /exit — Удаление бота из чата.

11. Генератор случайных чисел — От 1 до N.

12. Генератор паролей — Создание надежной строки с символами.

13. Конвертер валют — Актуальный курс через внешний API.

14. Погода — Точный прогноз в указанном городе.

15. QR-код генератор — Текст превращается в картинку QR-кода (qrcode, Pillow).

16. Сокращатель ссылок — Через сервисы вроде tinyurl.

17. Таймер — Напоминание через X минут.

18. Будильник — Установка оповещения на конкретное время.

19. Переводчик — Мгновенный перевод текста.

20. Калькулятор — Безопасное решение математических выражений в чате (simpleeval).

21. Поиск GIF — Поиск через Giphy API.

22. Викторина — Случайный вопрос с кнопками ответа.

23. Генератор мемов — Наложение текста на картинку.

24. Анализ текста — Подсчет слов и символов.

25. Камень-Ножницы-Бумага — Интерактивная игра с ботом.

26. Цитата дня — Случайная мотивирующая фраза.

27. Википедия-поиск — Получение краткой выжимки по запросу.

28. Конвертер видео в аудио — Извлечение звука через ffmpeg.

29. Стикер-пак инфо — Информация о выбранном стикере.

30. Генератор имен — Создание фэнтези-имен.

31. Уведомления о GitHub — Привязка Webhook для оповещений о пушах.

32. SQL-запросы — Выполнение простых SQL-команд (для админа).

33. Запуск кода — Выполнение Python-кода в песочнице.

34. Мониторинг сервера — Вывод нагрузки на CPU/RAM сервера (psutil).

35. Авто-ответчик (AI) — Интеграция с LLM (GroqCloud/Llama 3).

36. Распознавание речи — Аудио в текст.

37. Распознавание текста с фото — Подключение OCR API.

38. Авто-удаление спама — Фильтрация ссылок и мата.

39. Управление БД — White/Black листы пользователей через ORM.

40. Планировщик задач — Запуск заданий по cron-расписанию (apscheduler).

41. Встроенный Web App — Раздача статики через асинхронный aiohttp.web, работающий параллельно с пуллингом бота.

🚀 Установка и запуск

Клонируйте репозиторий:

git clone https://github.com/GeniusAllWorld/super-telegram-bot

cd super-telegram-bot

Создайте и активируйте виртуальное окружение:

python -m venv venv

# Для Windows:

venv\Scripts\activate

# Для Linux/macOS:

source venv/bin/activate

# После активации виртуального окружения

Установите зависимости:

pip install -r requirements.txt

Настройте переменные окружения:

Скопируйте файл .env.example и переименуйте его в .env.

Заполните все необходимые токены и API-ключи.

Запустите бота:

python main.py

(Для использования Уровня 5 не забудьте предварительно запустить локальный туннель, например: ngrok http 8080, и добавить полученную ссылку в файл .env).
